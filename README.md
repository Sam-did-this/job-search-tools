# Job Search Tools

Automated job search scripts for finding gigs and cash work on Craigslist.

## Features
- Searches multiple gig categories automatically
- Filters out unwanted job types
- Highlights priority listings (cash, daily pay, etc.)
- Tracks new vs. seen listings
- NYC-optimized with borough support

## Usage
```bash
python3 new-job-search.py
```

## Requirements
```bash
pip3 install requests beautifulsoup4
```
